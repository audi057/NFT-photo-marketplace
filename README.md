# NFT Photo MarketPlace


### Setup backend
1. Deploy contracts to private network of Ganache
```
(root directory)

$ npm run migrate:local
```

&nbsp;


### Setup frontend
1. NPM modules install
```
$ cd client
$ npm install
```

2. Execute command below in root directory.
```
$ cd ..
$ npm run client
```

3. Access to browser by using link 
```
http://127.0.0.1:3000
```
